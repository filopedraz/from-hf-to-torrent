## Seeding

### 1. Requirements and Clone models

```shell
pip install -r requirements.txt
```

```shell
mkdir data
cd ./data
git lfs install
huggingface-cli lfs-enable-largefiles .
git clone https://huggingface.co/gpt2
```
or modify and use [download.sh](./data/download.sh).

### 2. Change Parameters

Under [main.py](./main.py) change the following parameters as required:
```python
models = ["CodeLlama-34b-Instruct-hf", "StableBeluga2"]  # list of names of the cloned model folders you want to use
data_path = "./data/" # relative path where the models are stored
torrents_dir = "./torrents/" # relative path where the torrents will be stored
```

### 3. Set Environment Variables

For basic auth create a `.client_password` file containing just the password in it, at the root of this repository.
```shell
export TRANSMISSION_USER=prem # change this to your username
export TRANSMISSION_PASS=$(cat .client_password) # change this to your password
```

### 4. Start Transmission Client in the background

Modify absolute paths on [docker-compose.yml](./docker-compose.yml) file based on your machine.

```shell
docker compose up -d
```
WebUI should be accessible at `http://<localhost/public-ip>:9091`

> Note: Open Firewall if you are on a Mac - https://github.com/transmission/transmission/blob/main/docs/Port-Forwarding-Guide.md#open-your-local-firewall


### 5. Create the torrent and seed it

```shell
python main.py
```
